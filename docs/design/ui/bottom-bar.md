# 底部状态栏 (BottomBar)

> 索引：[UI INDEX](INDEX.md)
> 实现：`scripts/ui/bottom_bar.gd`

---

## 外观

固定在屏幕底部，三段式布局：

```
┌──────────────────┬──────────────────────────┬──────────────┐
│  💎 晶体: 350    │  选中: 3 战士 / 2 工人     │  战士 8 : 5  │
└──────────────────┴──────────────────────────┴──────────────┘
  资源段                    选中信息段                战况段
```

## 数据来源

| 区段 | 显示内容 | 数据来源 |
|------|---------|---------|
| 资源段 | `💎 晶体: {n}` | `hq_red.crystal` |
| 选中段 | `选中: {战士数} 战士 / {工人数} 工人`；无选中时隐藏 | `SelectionManager.selected_units` |
| 战况段 | `战士 {红} : {蓝}`（实时） | `UnitLifecycleManager` |

## 行为规则

- 始终可见（CanvasLayer 置顶）
- 选中段在无选中单位时**隐藏**，有选中时显示类型统计

## 断言验收

| 断言 | 验收标准 |
|------|---------|
| `bottom_bar_visible` | BottomBar 节点 `visible == true` |
| `hq_has_mesh` | HQ_red 有 MeshInstance3D（底栏需要 HQ 存在才有意义） |
