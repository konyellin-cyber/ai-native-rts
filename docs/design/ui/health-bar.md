# 单位血条 (HealthBar)

> 索引：[UI INDEX](INDEX.md)
> 实现：`scripts/fighter.gd` / `scripts/worker.gd` 内联

---

## 外观

每个单位头顶显示迷你血条：

```
  ████████░░  HP: 80/100
```

## 规格

| 属性 | 值 |
|------|----|
| 宽度 | 单位半径 × 3 |
| 高度 | 3px |
| 颜色 | 绿 (>60%) → 黄 (30–60%) → 红 (<30%) |
| 显示时机 | 受伤时显示；满血或 3 秒无新伤害后隐藏 |

## Headless 行为

不创建 TextureProgressBar 节点，逻辑层（hp 属性）正常运行。

## 断言验收

> 血条为视觉组件，当前无专属断言。满血隐藏 / 受伤显示的行为可在窗口模式截图中目视验证。
> 如需程序化验证，可在未来补充：检查单位受击后子节点中存在 ProgressBar。
